Privacy Policy

Last updated: 2026-05-21

This page describes how the QissaStudios content-production application handles data.

What we collect

The QissaStudios application is a personal content-production tool operated by a single individual ("the operator"). It does NOT collect, store, or process data from any user other than the operator.

Specifically, the application:

  • Connects to the operator's own Google account to access the operator's own YouTube channel (@QissaStudios) via the YouTube Data API and YouTube Analytics API.
  • Stores OAuth refresh tokens locally on the operator's personal device for the sole purpose of allowing the application to upload videos and read its own performance metrics.
  • Stores YouTube Analytics responses (aggregate metrics for videos uploaded by the operator on the operator's own channel) in a local SQLite database used only by the operator for internal analysis.

The application has no website-visitor surface, no user accounts, no signups, no comments, and no contact forms. There is nothing for an external user to interact with.

What we do NOT do

  • We do NOT collect data from any user other than the operator.
  • We do NOT share any data with third parties.
  • We do NOT use any tracking technologies.
  • We do NOT process payments.
  • We do NOT use cookies (the application is server-side; there is no public web frontend).

How we use the data we have access to

Data accessed through Google APIs is used solely to:

  1. Upload videos produced by the operator to the operator's own YouTube channel.
  2. Verify that those uploads completed successfully.
  3. Read the operator's own channel's performance metrics in order to inform future content decisions.

All such data is stored on the operator's personal device. It is not transmitted to any server, service, or person other than the operator's own Google account.

Google API data usage

This application's use of information received from Google APIs adheres to the Google API Services User Data Policy, including the Limited Use requirements.
